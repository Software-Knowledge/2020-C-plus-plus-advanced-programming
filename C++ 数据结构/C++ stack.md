c++ stack
---
1. 简要记录比较通俗易懂的stack(c++内置版本)
2. 头文件:`#include<stack>`
3. 是一个FIFO的线性链表

| 函数名  | 功能             | 其他备注       |
| ------- | ---------------- | -------------- |
| top()   | 取出栈顶元素     | 不删除栈顶元素 |
| pop()   | 删除栈顶元素     | 无             |
| empty() | 检验栈是否为空   | 为空则为true   |
| push()  | 在栈顶增加元素   | 无             |
| size()  | 返回栈中元素个数 | 无             |
